Edition #1: Using Markdown to create HTML emails with Maizzle

This is a arkdown starter project that will help you get started with creating newsletter email templates from Markdown files in Maizzle.

It supports all the framework features you already know and love:

  • Tailwind CSS
  • components
  • expressions
  • conditionals, loops etc.

Ordered lists

  1. Italics
  2. Bold text
  3. Links
